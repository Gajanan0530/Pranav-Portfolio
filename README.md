## 📘 Overview

Hey, I’m **Pranav**, a QA Test Engineer and Automation Tester. This portfolio is tailored for test-centric engineering and includes:

- QA role highlights and role rotation section
- Education, projects, and skills focused on software testing
- Interactive terminal (chatbot) with QA commands
- Visual QA style (bold, clean, and responsive)

---

## 🎯 Key Features

- **QA-first design**: profile centered around testing roles and outcomes
- **Interactive terminal**: ask `about`, `skills`, `projects`, `contact`
- **Skill cards**: automation, manual testing, QA tools, performance/security
- **Education timeline**: clear learning journey (B.Tech, class 12, class 10)
- **Project cards**: latest work, address book, Godot plugin
- **Contact badges**: GitHub/LinkedIn/LeetCode/CodeChef/Email

---

## 🛠️ Tech Stack

- Framework: React 18, TypeScript
- Styling: Tailwind CSS + shadcn/ui
- Build: Vite
- State & data: TanStack Query
- Form/Validation: React Hook Form + Zod

## ⚙️ Quick Start

```bash
git clone https://github.com/<your-github-username>/pranav-portfolio.git
cd pranav-portfolio
npm install
npm run dev
```

Open `http://localhost:8080` (or URL shown by Vite).

Build:

```bash
npm run build
```

## 🧩 Terminal Commands

- `help`: command list
- `about`: QA profile
- `skills`: QA stack
- `projects`: project list
- `contact`: reachout info
